This is a simple library management system written in Python.
It lets you manage books, authors, genres, and users. You can add, borrow, return, search for books, and display all books. 
You can also add, view, and display authors, genres, and users.


Make sure to configure the database connection in database_connection.py.

Run the program with python main.py. You'll be prompted to choose operations like adding a book, borrowing a book, etc.
Follow the prompts to perform actions.

The code is divided into different files for each operation type.
There's a main script to run everything and separate scripts for books, authors, genres, and users. 
Handle any database errors as they come up to keep everything running smoothly.
